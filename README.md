# prepare-project-for-llm

## Descrição (Português)

Este projeto tem como objetivo facilitar a preparação de projetos para análise por Inteligências Artificiais (IA), como LLMs (Large Language Models). Ao organizar, documentar e estruturar o projeto de forma clara, aumentam-se as chances de obter respostas mais precisas, rápidas e relevantes das IAs. O script automatiza tarefas essenciais, como limpeza de arquivos desnecessários, geração de documentação e padronização de estrutura, tornando o processo eficiente e menos suscetível a erros humanos.

### Por que este projeto é útil para análise por IA?

- **Reduz ruído**: Remove arquivos e pastas irrelevantes, focando apenas no que importa para a análise.
- **Padroniza estrutura**: Facilita o entendimento do projeto por humanos e máquinas.
- **Gera documentação**: Cria arquivos README e outros documentos essenciais automaticamente.
- **Acelera o processo**: Automatiza etapas repetitivas, economizando tempo.
- **Melhora a precisão**: Fornece contexto claro para a IA, resultando em respostas melhores.

## Como usar

1. Execute o script `prepare-project-for-llm.js` na raiz do seu projeto.
2. Siga as instruções exibidas no terminal.
3. Pronto! Seu projeto estará otimizado para análise por IA.

## Description (English)

This project aims to make it easier to prepare codebases for analysis by Artificial Intelligences (AI), such as LLMs (Large Language Models). By organizing, documenting, and structuring the project clearly, you increase the chances of getting more accurate, faster, and relevant answers from AIs. The script automates essential tasks like cleaning unnecessary files, generating documentation, and standardizing structure, making the process efficient and less prone to human error.

### Why is this project useful for AI analysis?

- **Reduces noise**: Removes irrelevant files and folders, focusing only on what matters for analysis.
- **Standardizes structure**: Makes the project easier to understand for both humans and machines.
- **Generates documentation**: Automatically creates README and other essential documents.
- **Speeds up the process**: Automates repetitive steps, saving time.
- **Improves accuracy**: Provides clear context for the AI, resulting in better answers.

## How to use

1. Run the `prepare-project-for-llm.js` script at the root of your project.
2. Follow the instructions shown in the terminal.
3. Done! Your project will be optimized for AI analysis.
